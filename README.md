# MJK LAB - Site V4

Site catálogo da MJK LAB com painel para cadastrar produtos, links da Shopee, Mercado Livre e WhatsApp.

## Arquivos para enviar ao GitHub

- index.html
- estilo.css
- script.js
- logo-mjk.jpg
- README.md

## Importante

Os produtos cadastrados ficam salvos no navegador pelo localStorage. Use o botão "Backup dos produtos" antes de trocar de computador, limpar o navegador ou fazer grandes alterações.
